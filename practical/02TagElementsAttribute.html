Tag vs Element vs Attribute

Tag

Which comes in < AngularBrackets>, 90% tags are come with starting Tag and Closing Tag, like h1, p , head, body, title etc.

10% tags are self Closing tag or Empty Tag or Void Tag like, br, hr, img, input etc.

Element

Start Tag - Some Content - End Tag

Attribute

They provide some additional information by which behaviour of element will change .

Always Written in start tag
